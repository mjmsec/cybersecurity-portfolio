# Linux Basics

## Check Current Directory

```bash
pwd
```

## List Files

```bash
ls
```

## Change Directory

```bash
cd directory_name
```

## Create Directory

```bash
mkdir test
```

## Create File

```bash
touch file.txt
```

## Change Permissions

```bash
chmod 755 file.txt
```

## Change Ownership

```bash
chown user:user file.txt
```
