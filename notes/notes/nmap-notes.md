# Nmap Notes

## Basic Scan

```bash
nmap target-ip
```

## Service Detection

```bash
nmap -sV target-ip
```

## OS Detection

```bash
nmap -O target-ip
```

## Aggressive Scan

```bash
nmap -A target-ip
```

## Scan Specific Ports

```bash
nmap -p 80,443 target-ip
```

## Learning Goal

Understand host discovery, service detection, and network enumeration in authorized environments.
